GitHub Pages

Opal RM Prototype documentation

This repository publishes multiple versions of its documentation so reviewers can see the latest, branch-specific, and pull-request-specific content without waiting for a merge to main.

Standard locations

  • Latest documentation from main
  • branches/<branch-name>/ for branch previews
  • pulls/pr-<number>/ for pull request previews

Examples

  • branches/feature-name/ for a feature branch preview
  • pulls/pr-123/ for a pull request preview
